AtmaFM - Underground Online Radio

I have always been a huge fan of music. Unfortunatelly, my musical taste is quite a peculiar one and so I always had a hard time finding a radio station that would fit my needs. That is why I have build my own. A personal online radio station that focuses primarily on ambient & related genres.

It is written in Python using Flask framework, running on Fedora Linux, streaming nonstop 24/7.

Covid Tracker - Real-Time Data Visualisation for USA

The pandemy of Corona virus has been a huge topic for quite some time. I did not really believed in all the stories and all the numbers associated with it. Especially when USA started to scream. A little on good API and this Covid TrackBoard came into existence.

It is written in Python Dash, a nice framework for browser based data visualisation. I really like Dash, although it is quite a pain to style the HTML from within the Python code.

SC ACSIL C++ Custom Studies for MySQL

Working with Sierra Chart trading platform and coding custom studies and trading systems using its ACSIL C++ framework is quite a terrible experience, especially when you need these to be connected to MySQL database. Here I put together a working example of a simple study that writes details of each entry into a database table. Voila. Seems like a trivial task, but no. It was quite a headache. Believe me. Follow the README.md EXACTLY if you want to make it work, otherwise you will loose all your hair when trying to fix the bugs.

It is written in C++ using on indows using Visual Studio.
